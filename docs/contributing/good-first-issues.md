# Good First Issues

Welcome! We love contributions. Here are some areas where you can get started.

## Finding Issues
Look for the `good-first-issue` label on our [GitHub Issues](https://github.com/OSS-AI-Agent-Tool/OSS-AI-Agent-Tool/issues).

## Suggested Areas

### Documentation
- Fix typos or clarify confusing sections.
- Add examples to the [CLI Reference](./reference/cli.md).
- Improve [Runbooks](./runbooks/README.md).

### Agents
- Create a new agent profile using `aidt new-agent`.
- Add unit tests for existing agents.

### GUI
- Improve accessibility (ARIA labels).
- Add tooltips for complex configuration options.

## Getting Started
1. Read [CONTRIBUTING.md](../CONTRIBUTING.md).
2. Fork the repo and clone it.
3. Run `npm install` and `npm test` to ensure everything is working.
4. Pick an issue and comment that you're working on it.
